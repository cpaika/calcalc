## Cal Calc
This project calculates your busyness over the years by reading from google calendar.

### Prerequisites
Golang 1.13+ installed
On a mac, you can install golang via [Homebrew](https://brew.sh/):
`brew install go`

### Running the application
1.  Follow the instructions [on this page](https://developers.google.com/calendar/quickstart/go) and download the credentials.json file.  Put it in the calcalc folder.
2.  `go run main.go`
3.  Follow the prompt and enter an access token
